## Finance Simulator (Python) ##

Pequeno projeto desenvolvido para praticar fundamentos de Python na prática, simulando a viabilidade de um financiamento imobiliário.

### Conceitos aplicados:
- Funções e parâmetros
- Validação de entrada do usuário (`while` + `try/except`)
- Cálculo de juros compostos
- Organização e legibilidade de código
- Separação de responsabilidades entre funções

O programa considera:
- valor do imóvel
- valor da entrada
- salário
- prazo do financiamento
- taxa de juros

Mais importante do que o resultado final foi o processo de aprendizado e organização do código.

Este projeto faz parte da minha jornada de estudos em Python.
